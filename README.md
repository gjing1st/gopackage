#
## 🌱1基本介绍
### 1.1 项目目的
为方便go语言开发，将常用函数进行封装
### 1.2 项目简介
基于以上目的，每个包都单独封装，可以单独导入包使用，部分函数使用了gf框架中的函数

## 2 使用说明
```bash
go get -u -v github.com/gjing1st/gopackage
```

#### 推荐使用 go.mod:
```bash
require github.com/gjing1st/gopackage latest
```

## 3. 限制
```bash
golang版本 >= 1.14
```
